# Components Directory

React components organized by feature.

## Structure

- `ui/` - Base UI components from shadcn/ui (Button, Card, Dialog, etc.)
- `auth/` - Authentication components (LoginForm, RegisterForm, etc.)
- `course/` - Course management components
- `file/` - File management components
- `reader/` - PDF reader components
- `learn/` - AI tutor components (explanation, quiz, Q&A)
- `admin/` - Admin dashboard components
- `layout/` - Layout components (Navigation, Header, etc.)
- `quota/` - Quota display components
- `settings/` - Settings page components

## Guidelines

- Use shadcn/ui components as base building blocks
- Keep components small and focused
- Use TypeScript for all components
- Export from index.ts files for clean imports
